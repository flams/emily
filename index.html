What is Emily? | Examples | Documentation

Emily is a JavaScript Framework. It offers convenient tools for building MVC applications.
It has the following requirements:

  • Emily should use only standard technologies.
  • Emily should let native features be usable out of the box and not re-implement them
  • Emily should use the new HTML5 features when they're the best way to achieve something
  • Emily should be 100% TDD with 100% code coverage
  • Emily's modules should follow the commonJS AMD
  • Emily should run in any JavaScript runtime
